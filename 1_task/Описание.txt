1. Команда ls - покажет содержимое директории. Опция -a покажет все файлы(скрытые и наоборот). Опция -l покажет права доступа.
Команда ll также покажет все файлы в директории с правами доступа.
2. Команда grep 404 access.log покажет все упомнинания со строкой 404.Команда cat access.log выведет значение на консоль.
3. Команда cat 404.html | grep 404 > 403.html | sed -i 's/404/403/g' 403.html копирует и изменяет исходный файл. Сначала вывожу командой cat содержимое файла, затем перенаправляю команде grep, ищу в нем строку 404 и записываю в выходной файл. Затем перенаправляю команде set и в нем указанную строку меняю на другую с флагом g(для изменения всех совпадающих строк).
4. При создании файла index.html по стандарту права доступа для владельца и группы на чтение и запись, а для остальных только на чтение. Таким образом для выполнения задания были применены след команды: chmod g-w index.html, chmod o-r index.html.
5. Команда ln -s index.html index2.html для создания символической ссылки. Опция -s для создания симлинков.
6. Команда ll для вывода всех файлов и папок в папке var
7. Команда  echo "<html> <head> <title> 503 </title> </head> </html>" > 503.html
для создания файла 503.html. Стандартной командой echo которая принимает аргументы, перенаправляю данные в новый файл который будет создан.
