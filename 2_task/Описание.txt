Для выполнения этого задания делаем имитацию командой git clone ssh://git@example.com:example/test.git. Затем заходим в этот проект (test) "cd test". Командой git branch узнаем на какой ветке находимся. Ветка master. Переключаемся на ветку: git checkout dev. Создаем файл Readme.md командой touch Readme.md. Командой echo "### May be in future" > Readme.md записываем текст в файл. Проверяем командой cat Readme.md записались ли данные. Записались. Затем командой git add Readme.md добавляем файл в гит. Проверяем добавился и отслеживает ли гит этот файл командой git status. После командой git commit -m "Add Readme.md file" делаем коммит. Затем командой git push origin dev заливаем изменения на ветку dev.
